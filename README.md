# IoT-Based Cold Chain Monitoring System
## Overview
The IoT-Based Cold Chain Monitoring System is designed to monitor temperature and humidity conditions in storage environments such as vaccine storage, food transportation, and pharmaceutical cold chains. The system uses an ESP32 microcontroller and a DHT sensor to collect environmental data and display it in real time.
## Objective
The main objective of this project is to ensure that temperature-sensitive products are stored and transported under safe conditions by continuously monitoring environmental parameters.
## Components Used
- ESP32 Microcontroller
- DHT11 / DHT22 Temperature and Humidity Sensor
- OLED Display
- Breadboard
- Jumper Wires
- Power Supply (USB)
## Working Principle
1. The DHT sensor measures temperature and humidity.
2. ESP32 reads the sensor data.
3. The collected data is processed by the microcontroller.
4. The OLED display shows real-time values of temperature and humidity.
## Features
- Real-time monitoring
- Low-cost IoT solution
- Easy to implement
- Suitable for small-scale cold storage monitoring
